## 🚀 Overview

A **full-stack AI-powered hospital management system** designed to handle real-world hospital operations efficiently.

It integrates:
- Real-time patient tracking  
- Smart bed allocation  
- Billing system  
- AI-based risk prediction  

---

## 🔥 Features

- 🛏 **Bed Allocation** (ICU, General, Emergency)
- 👨‍⚕️ **Patient Management Dashboard**
- 💊 **Expense & Billing System**
- 🧠 **AI Risk Prediction (Low / Medium / High)**
- 🔁 **Auto Bed Reallocation**
- 📊 **Real-time Hospital Overview**

---

## 🧠 AI Risk Prediction

This system includes a Machine Learning model that predicts patient risk levels:

- 🔴 High Risk  
- 🟡 Medium Risk  
- 🟢 Low Risk  

👉 Helps hospitals prioritize critical patients and improve decision-making.

---

## 🛠 Tech Stack

- **Frontend:** React (Vite)  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **AI/ML:** Python  
- **Deployment:** Vercel + Render
