## ✨ Features

### 👨‍⚕️ Patient Management

* Add, delete, discharge patients
* Search & filter patients
* Track disease, age, and status

---

### 🛏 Bed Management

* Real-time bed visualization
* Occupied vs available beds
* Department-wise filtering

---

### 🧠 AI Prediction System

* Predicts patient risk: **Low / Medium / High**
* Provides recommendations:

  * ICU Required
  * Regular Monitoring
  * Normal Care
* Built using **Random Forest Algorithm**

---

### 💰 Smart Billing System

* Room rent (per day)
* Doctor fees
* Medications
* Tests & lab charges
* ICU & emergency charges
* Surgery costs
* Nursing care
* Manual extra expenses
* GST-ready billing

---

### 💊 Pharmacy Management (NEW 🚀)

* Pharmacy role login
* Manage medications assigned to patients
* Track medicine-related expenses
* Integrated with billing system
* Supports real hospital workflow

---

### 🔐 Multi-User Role System

| Role         | Access                      |
| ------------ | --------------------------- |
| 👑 Admin     | Full system control         |
| 👨‍⚕️ Doctor | View patients + AI insights |
| 💼 Reception | Add patients + billing      |
| 💊 Pharmacy  | Manage medicines + expenses |

---

### 📊 Dashboard & Analytics

* Total / occupied / free beds
* Patient statistics
* Visual charts & insights
